# BMI-Calculator
A simple Python application that calculates Body Mass Index (BMI) using a user's height and weight. The program validates user input, displays the BMI value, and classifies it into standard BMI categories such as underweight, normal weight, overweight, and obesity.
 
## Features

- Accepts user name, weight, and height as input
- Calculates BMI using the standard formula
- Displays BMI up to 2 decimal places
- Categorizes BMI into:
  - Underweight
  - Normal weight
  - Overweight
  - Obese
  - Severely obese
  - Morbidly obese
- Validates that height and weight are positive values

## BMI Formula

```text
BMI = Weight (kg) / Height² (m²)
```
